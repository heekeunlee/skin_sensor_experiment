## 세미나 1주차: "피부가 뭔데?" — 피부 구조 기초

### 🎯 이번 주 목표
피부가 왜 층상 구조인지, 수분이 왜 중요한지 이해한다.

### 📖 핵심 개념 (쉬운 말로)

**피부의 3층 구조**

```
[피부 바깥 → 안쪽]

┌─────────────────────────────┐  ← 각질층 (stratum corneum)
│  죽은 세포 + 기름으로 방어막  │     두께: 머리카락 1/10 굵기 (10~20㎛)
│  ★ 수분이 여기 있어야 촉촉함  │     이 실험에서 측정하는 부분!
└─────────────────────────────┘
┌─────────────────────────────┐  ← 표피층 (epidermis)
│   살아있는 세포들이 층층이    │     두께: 약 0.1mm
└─────────────────────────────┘
┌─────────────────────────────┐  ← 진피층 (dermis)
│  혈관, 신경, 콜라겐이 있는 곳 │     두께: 약 2mm
└─────────────────────────────┘
```

> **비유:** 피부는 마치 페인트칠한 벽 같다.  
> 가장 바깥 페인트 층(각질층)이 수분을 잡아두는 역할을 한다.  
> 페인트가 벗겨지면(건성 피부) 안쪽 벽이 직접 공기에 노출되어 문제가 생긴다.

### 🔍 수분이 왜 중요한가?

| 각질층 수분 상태 | 피부 상태 | 비유 |
|:---:|:---:|:---:|
| 충분 (수분 > 30%) | 탄탄하고 매끄러움 | 충분히 불은 건포도 |
| 부족 (수분 15–30%) | 약간 당김 | 반쯤 마른 건포도 |
| 매우 부족 (< 15%) | 각질 발생, 가려움 | 완전히 마른 건포도 |

## 세미나 2주차: "우리 센서의 원리" — 전기와 자기 기초

### 🎯 이번 주 목표
우리가 사용할 센서가 어떻게 피부 수분을 감지하는지 원리를 이해한다.

### 📖 핵심 개념 (쉬운 말로)

**전기와 물의 관계**
- 물(H₂O)은 전기를 약간 전달한다.
- 소금물은 더 잘 전달한다 (전해질 때문).
- 피부에 수분이 많으면 → 전기가 더 잘 통한다.
- 우리 센서는 이 성질을 이용한다.

**맴돌이전류(Eddy Current)란?**
> 쉬운 말: "자기장이 도체 근처에 가면, 도체 안에 소용돌이 전류가 자동으로 생긴다"

```
[실험으로 이해하기]

코일(전선을 돌돌 감은 것)에 전기를 흘리면
        ↓
코일 주변에 자기장이 생긴다 (마치 자석처럼)
        ↓
그 자기장이 피부(도전성 물질) 근처에 닿으면
        ↓
피부 안에 소용돌이(맴돌이) 전류가 유도된다
        ↓
이 전류는 피부 수분 함량에 따라 세기가 달라진다
        ↓
코일의 전기 특성(주파수)이 변한다 → 이걸 측정!
```

**Skin Effect(피부 효과)란?**
> 주파수가 높을수록 전류는 표면에만 흐른다.

| 주파수 | 전류가 흐르는 깊이 | 측정 가능한 층 |
|:---:|:---:|:---:|
| 낮음 (1 MHz) | 깊이 침투 | 진피층까지 |
| 높음 (100 MHz) | 표면만 통과 | 각질층만! ← 우리가 원하는 것 |

> **비유:** 라디오 주파수가 높을수록 건물에 잘 막히듯,  
> 높은 주파수 전류는 깊이 못 들어가고 피부 표면에서만 맴돈다.

### 🔬 우리 센서의 작동 원리 한 줄 요약
```
코일 → 고주파 자기장 발생 → 각질층에 맴돌이전류 유도 → 
수분 많으면 전류↑ → 코일 주파수 변화 → 수분% 계산
```

## 세미나 3주차: "실험의 언어" — 데이터와 통계 기초

### 🎯 이번 주 목표
측정값을 어떻게 정리하고, "이 센서가 정확하다"를 어떻게 증명하는지 이해한다.

### 📖 핵심 개념 (쉬운 말로)

**In-Vitro vs. In-Vivo vs. 임상실험**

```
In-Vitro (인 비트로): "시험관 안에서" = 실제 사람 피부 없이, 
                       비슷한 재료로 먼저 테스트
                       → 예: 소금물로 센서가 작동하는지 확인

In-Vivo (인 비보):    "살아있는 것에서" = 본인들 피부에 직접 테스트
                       → 예: 팀원 전완부에 센서 가져다 대기

임상실험 (Clinical):  정해진 규칙 아래 30명에게 정식으로 실험
                       → 논문에 넣는 공식 데이터
```

**평균(Mean)과 표준편차(SD)**

> **비유:** 양궁 과녁에 화살 10발을 쏜다고 생각하자.
> - 평균 = 화살들의 "중심 위치"  
> - 표준편차 = 화살들이 얼마나 퍼져 있는가 (작을수록 정확!)

**상관계수 R²란?**

> 두 측정값이 얼마나 같은 경향을 보이는지 나타내는 숫자.  
> - R² = 1.0: 완벽히 일치 (불가능에 가까움)  
> - **R² > 0.85: 우리 목표 (아주 강한 일치)**  
> - R² = 0.0: 전혀 관계없음

**Bland-Altman Plot이란?**

> "두 기기로 같은 것을 쟀을 때 얼마나 차이나는지"를 그래프로 표현.  
> 우리는 비접촉 센서와 Corneometer(표준장비)를 비교 → 차이가 작을수록 좋다.

## 📋 실험 I-A: 소금물 농도 vs. 센서 반응 실험

### 목적
소금물 농도(= 전도율)가 달라지면 센서 주파수가 어떻게 변하는지 확인한다.  
→ **"더 짠 물 = 더 촉촉한 피부"를 모사**

### ⚙️ 센서 조립 순서 (학생 A 담당)

> **멘토와 함께 첫 번째 조립은 반드시 같이 진행할 것**

**Step 1: 코일 제작**
```
에나멜 동선을 지름 20mm 원형 틀에 20번 감는다
(연필이나 20원짜리 동전을 틀로 사용 가능)
끝부분 2cm는 감지 않고 리드선으로 남긴다
사포로 끝부분 에나멜 코팅 살짝 벗겨 납땜 준비
```

**Step 2: LC 발진기 회로 조립**
```
브레드보드에 74HC14 IC 꽂기
→ 핀 배치는 첨부 회로도 그림 참고 (별첨 1)

**Step 3: Arduino 연결 및 코드 업로드**
```

### 📊 결과 분석 (학생 B 담당)

엑셀에서:
1. 각 농도별 **평균 주파수** 계산
2. X축 = 소금물 농도(%), Y축 = 평균 주파수 → **산점도** 그리기
3. 추세선(Trendline) 추가 → **R² 값 확인**

**기대 결과:**
> 농도가 높을수록 주파수가 낮아지는 경향이 보여야 한다.  
> (더 짠 물 → 전도율 높음 → 맴돌이전류 강해짐 → 코일 주파수 감소)

## 📋 실험 I-B: 거리 의존성 실험 (Lift-off 실험)

### 왜 필요한가?
> 실제 측정 때 손이 떨리면 센서와 피부 거리가 달라진다.  
> 거리가 달라질 때 신호가 어떻게 변하는지 미리 파악해야 보정이 가능하다.

### 📋 절차

```
1. 컵 3번 (0.5% 생리식염수)를 기준으로 사용
2. 센서-컵 표면 거리를 1cm부터 5cm까지 변화
   - 1cm / 1.5cm / 2cm / 3cm / 4cm / 5cm
3. 각 거리마다 5회 측정 → 기록
4. 엑셀에서 거리 vs. 주파수 그래프 그리기
```

**기대 결과:**
> 거리가 멀어질수록 주파수 변화가 줄어든다.  
> 2–3 cm 이내가 측정 가능한 유효 범위임을 확인한다.

## 📋 실험 I-C: 보습제 흡수 모사 실험

### 목적
화장품(보습제)을 바르기 전/후 차이를 모사한다.

```
[실험 설정]
젖은 티슈(수분 많음) → 센서 측정
젖은 티슈 위에 바셀린 0.1g 도포 → 다시 측정
같은 부위를 T+1분, T+5분, T+10분에 반복 측정

→ "보습제 흡수 곡선" 관찰
```

> **이게 왜 중요한가?**

# 🧑‍🔬 PART 2: In-Vivo 파일럿 실험 (Week 7–9)

## ✅ 이것이 왜 필요한가?
> In-Vitro에서 소금물로 확인했다면, 이제 **실제 살아있는 피부에 테스트**한다.  
> 피험자는 팀원 3명 본인들 + 가족/친구 7명으로 총 10명.

## 📋 측정 전 준비 사항

### 환경 체크리스트 (매 실험 전 확인)
```
☐ 실내 온도: 20~22°C 인지? (에어컨/히터 조절)
☐ 실내 습도: 45~55% 인지? (스마트폰 앱 또는 습도계 확인)
☐ 측정 시간: 오전 9시~오후 1시 사이인지?
☐ 피험자: 측정 전 30분 실내 착석 대기했는지?
☐ 측정 부위: 세척 후 10분 경과했는지?
```

### 측정 부위 세척 방법
```
1. 일반 비누(향료 없는 것)로 측정 부위(팔 안쪽) 세척
2. 흐르는 증류수로 헹굼 (수돗물 X → 미네랄이 있어서)
3. 깨끗한 종이 타월로 가볍게 두드려 건조
4. 10분 대기 → 피부 온도 안정화
```

## 📋 실험 II-A: 기저값 측정 (Baseline)

### 10명 기저 수분 측정

**측정 순서 (1인당 약 15분):**

```
Step 1. 피험자 착석 (팔을 테이블에 수평으로 올려놓기)
Step 2. 학생 A: 센서 고정 거리 2cm 유지하며 측정
        → 학생 B: 아두이노 값 3회 읽어서 기록
Step 3. 학생 C: Corneometer(표준 장비, 멘토 제공) 동일 부위 5회 측정
Step 4. 피부 온도 기록 (MLX90614 센서)
Step 5. 다음 피험자로 교체
```

**기록지 예시:**

```
피험자 코드: P001  나이: __세  성별: __  날짜: __________

[환경]  온도: __°C  습도: __%

[비접촉 센서] 측정1=___ 측정2=___ 측정3=___ → 평균: ___
[Corneometer] 측정1=___ 측정2=___ 측정3=___ 측정4=___ 측정5=___ → 평균: ___
[피부온도]    ___°C

피부 타입 자가 평가: □건성  □정상  □지성  □복합성
```

## 📋 실험 II-B: 보습제 도포 전후 비교

### 실험 설계

```
[한 팔을 3구역으로 나눈다]

|--- 구역 A ---|--- 구역 B ---|--- 구역 C ---|
   (바셀린)     (세타필크림)    (아무것도 X)
                                ↑ 대조군(Control)
```

**시간대별 측정:**

| 시점 | 비접촉 센서 | Corneometer | 메모 |
|:---:|:---:|:---:|:---:|
| T=0 (도포 전) | ✅ | ✅ | 기저값 |
| T=1분 | ✅ | ✅ | 즉각 반응 |
| T=5분 | ✅ | ✅ | 초기 흡수 |
| T=10분 | ✅ | ✅ | 중기 |
| T=30분 | ✅ | ✅ | 장기 보습 |

> **💡 주의:** 보습제 도포는 0.1g이다. (쌀알 크기만큼)  
> 학생 C는 도포 후 일정하게 1분간 부드럽게 문지르기.

## 📊 Phase II 데이터 분석 방법 (쉽게 따라하기)

### Step 1: 엑셀에 데이터 정리

```
A열: 피험자 코드 (P001~P010)
B열: 비접촉 센서 평균값
C열: Corneometer 평균값
D열: 피부 온도
E열: 측정 날짜
```

### Step 2: 산점도로 상관관계 확인

```
X축 = Corneometer 값 (기준 장비)
Y축 = 비접촉 센서 값 (우리 센서)

둘이 비슷하다면 → 데이터 점들이 사선으로 나열됨
```

### Step 3: 추세선 + R² 값 확인

> 엑셀에서: 차트 클릭 → 추세선 추가 → "수식 및 R² 값 표시" 체크  
> R² 가 0.7 이상이면 좋은 신호!

### Step 4: 시간대별 수분 변화 그래프

```
X축 = 시간 (0, 1, 5, 10, 30분)
Y축 = 수분값

선 3개 그리기: 바셀린 / 세타필 / 대조군
→ 어떤 제품이 수분을 더 오래 유지하는지 시각적으로 비교
```

# 🏥 PART 3: 임상 검증 실험 (Week 10–12)

## ✅ 이것이 왜 필요한가?
> 팀원과 지인 10명은 너무 적다. 논문에 넣으려면 **30명**이 필요하다.  
> 다양한 피부 타입의 사람들을 측정해야 센서의 신뢰성을 증명할 수 있다.

## 📋 30명 모집 전략

| 모집 방법 | 목표 인원 | 방법 |
|:---:|:---:|:---|
| 학교 커뮤니티 앱(에브리타임 등) | 15명 | "피부 수분 무료 측정" 모집 공고 |
| 지인/가족 | 10명 | 직접 연락 |
| 학과 교수/조교 | 5명 | 협조 요청 |

## 📋 동의서 준비 (IRB 면제 범위 내 비침습 실험)

학생 C가 작성할 동의서 필수 항목:
```
1. 실험 목적 설명 (비접촉 피부 수분 측정 기기 개발 연구)
2. 측정 방법 (피부에 닿지 않는 센서, 약 15분 소요)
3. 개인정보 보호 (익명 처리, 연구 목적 외 사용 금지)
4. 참여 철회 자유 (언제든 거부 가능)
5. 서명란
```

> **측정 순서는 동일**: 기저 측정 → 보습제 도포 → T+1, 5, 10, 30분 측정

# 🚨 자주 하는 실수 & 해결법

| 실수 | 해결법 |
|:---:|:---|
| 측정할 때마다 거리가 달라짐 | 스탠드로 코일 고정 후 측정 |
| 팔이 움직여서 값이 튐 | 팔꿈치 아래 쿠션 받쳐 고정 |
| 데이터를 기록 안 하고 지워버림 | 측정 즉시 종이에 기록, 나중에 엑셀 입력 |
| 온도/습도를 안 쟀음 | 체크리스트 맨 위에 고정 항목으로 두기 |
| 피험자가 보습제를 미리 발라옴 | 측정 전날 공지: "내일 측정 전 화장 금지" |
| R² 값이 너무 낮게 나옴 | 이상값(outlier) 제거 후 재분석, 멘토 상담 |
